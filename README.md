# ISLAMU I-VSD Thesis and Agentic Skill

This repository contains the academic thesis materials, reusable I-VSD framework materials, and supporting agentic skill files.

The thesis and framework materials are shared to support learning, citation, review, and reuse by Muslim software developers, researchers, educators, and islamic value sensitive design practitioners.

Please cite:

Amir Akrari / ISLAMU ASBL. "Islamic Value-Sensitive Design (I-VSD): A Moral Framework for Provider Mediated Software Services." 2026.
